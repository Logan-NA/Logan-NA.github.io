4.6.1、方法的重载

概念

在同一个类中，允许存在一个以上的同名方法，只要他们的参数个数或者参数类型不同即可

变量的复制

​ 如果变量是基本数据类型，此时赋值的是变量所保存的数据值

​ 如果变量是引用数据类型，此时复制的是变量所保存的数据的地址值

**方法的形参的传递机制、值传递

1、形参：方法定义时，声明的小括号内的参数

​ 实参：方法调用时，时机传递给形参的数据

2、值传递机制

​ 如果参数是基本数据类型，此时实参赋给形参是实参真实存储的数据值

​ 如果参数是引用数据类型，此时实参赋给形参的是实参存储数据的地址值
